# Atlas Helper Chrome Extension

Automatically import orders from 12+ retailers into Atlas Resell Hub.

## Supported Retailers
Amazon, Walmart, Target, Best Buy, Woot, eBay, Costco, Home Depot, Newegg, Wayfair, Etsy, Chewy

## Download
[Download Latest ZIP](https://github.com/dabdelhamid1126-code/atlas-helper-extension/archive/refs/heads/main.zip)

## Installation
1. Download and extract the ZIP
2. Go to `chrome://extensions/`
3. Enable Developer Mode
4. Click "Load unpacked" and select the `extension` folder
